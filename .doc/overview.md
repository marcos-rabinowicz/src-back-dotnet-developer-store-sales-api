# Overview

Este repositório implementa a **Store Sales API**, com foco em domínio de **Vendas** e regras de negócio de descontos por quantidade.  
A avaliação enfatiza: DDD, separação de camadas, EF Core/PostgreSQL, testes unitários e de integração, e clareza de código.

Principais habilidades exercitadas:
- C#/.NET 8, DDD + Clean
- Regras de negócio e Domain Events
- EF Core (migrations, mapeamentos), PostgreSQL
- MediatR (handlers) e AutoMapper (profiles)
- Testes unitários/integration (xUnit, FluentAssertions, Bogus, NSubstitute)
- Paginação, ordenação, filtros e tratamento de erros
